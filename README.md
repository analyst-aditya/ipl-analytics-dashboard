# 🏏 IPL Analytics Dashboard

An interactive and visually engaging IPL Analytics Dashboard developed using HTML, CSS, JavaScript, and Chart.js. This project provides detailed insights into IPL statistics from 2008–2025, including team performance, player records, season analysis, and championship trends.

---

## 🚀 Live Demo
🔗 https://analyst-aditya.github.io/ipl-analytics-dashboard/

---

## 📌 Features

- 📊 Interactive Data Visualizations
- 🏆 Team Titles & Win Percentage Analysis
- 🟠 Orange Cap Statistics
- 🟣 Purple Cap Statistics
- 📈 Runs & Matches Analysis by Season
- 📅 IPL Champions Timeline
- 👥 Top Batsmen & Bowlers Comparison
- 🎨 Modern Responsive Dashboard UI
- ⚡ Dynamic Season Filter Functionality

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
